# 📱 Engañón - Impostor con Arte Andaluz (TFG)

### 🎭 ¿De qué trata el proyecto?
**Engañón** es una aplicación nativa de Android que reinventa el clásico juego de rol tipo "Impostor". La clave del proyecto es su **personalización temática basada en el léxico andaluz**, aportando una experiencia de usuario divertida, cercana y única. 

Es el resultado de mi **Trabajo Fin de Grado (TFG)**, donde he volcado mis conocimientos en desarrollo móvil y mi visión de marketing para crear un producto con identidad propia.

### 🛠️ Stack Tecnológico
* **Lenguaje:** Java ☕ (Android nativo)
* **IDE:** Android Studio
* **Interfaz:** XML con diseño adaptativo (Material Design)
* **Lógica:** Gestión de estados de juego y roles en tiempo real.

### 🚀 Características principales
* **Localismo y Gamificación:** Uso de expresiones andaluzas para definir roles y acciones.
* **Interfaz intuitiva:** Diseñada para una curva de aprendizaje mínima.
* **Arquitectura:** Código organizado para facilitar futuras actualizaciones de léxico o modos de juego.

### 🧠 El reto técnico
Lo más interesante de este proyecto fue gestionar la **dinámica de grupos** dentro de la app y asegurar que el diseño visual mantuviera la coherencia con el tono humorístico del léxico utilizado sin perder la profesionalidad de una app nativa.
